# Fake News Detection (ML Project)

## Overview

This project detects whether a news article is real or fake using machine learning. It uses basic NLP techniques to process the text and a classification model to make predictions.

## Tech Used

* Python
* Pandas, NumPy
* Scikit-learn
* NLTK
* Google Colab

## What I Did

* Cleaned and preprocessed text data
* Converted text into numerical form using TF-IDF
* Trained a classification model
* Tested the model on unseen data

## How to Run

1. Open the notebook in Google Colab
2. Run all cells step by step

## Result

The model can classify news articles with good accuracy.
